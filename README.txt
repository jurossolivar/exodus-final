To install exodus:

1. Download both zip files
2. extract both zip files
3. copy all the contents on the 'exe file' to the main exodus file
4. run exodus.exe
5. enjoy

To open the source code:

[IMPORTANT!!!] Use Godot version 3.5 to open the project
(using godot 4 will mess the project up)

1. open godot
2. press scan
3. find the main exodus file you have recently downloaded and extracted
4. open
